Descripción General y Objetivos

Este proyecto consiste en la construcción de un vehículo robótico de bajo costo diseñado para operar de forma autónoma a largas distancias utilizando la red móvil **4G**. El carro es capaz de navegar, evadir obstáculos y transmitir datos de telemetría y video en vivo a un dashboard remoto.

Objetivos Principales

  • Construir un vehículo robótico capaz de moverse de forma autónoma.
  • Implementar un sistema robusto de evitación de obstáculos (**ToF** - *Time of Flight*).
  • Establecer un canal de comunicación en tiempo real (video + datos) con una laptop a una distancia superior a 500 metros mediante tecnología **4G**.
  • Desarrollar un dashboard interactivo para el monitoreo y registro de la misión.

Comunicación y Protocolo

La **Raspberry Pi** actúa como servidor usando **Flask** y **SocketIO** para enviar los datos por **WebSocket** en tiempo real. El cliente en la laptop (dashboard **Tkinter**) recibe un paquete **JSON** de telemetría cada segundo (1 Hz).

Modo de Funcionamiento y Software

Flujo de Operación

  1. Inicio: El carro se enciende, la Pi se conecta automáticamente a Internet vía el Módulo GSM 4G.
  2. Servidor y Lectura: El servidor Flask inicia y la Pi comienza a leer datos del ToF, GPS y batería (1 Hz).
  3. Monitoreo: La Laptop se conecta al servidor. Los datos de sensores, el mapa GPS y el video se actualizan en el dashboard.
  4. Autonomía y Decisión: El carro avanza. Si el Sensor ToF detecta un obstáculo (< 100cm), la **Raspberry Pi** toma la decisión de detenerse y cambiar la dirección automáticamente, sin intervención humana.

Software Utilizado

El núcleo del proyecto es **Python 3**, garantizando un desarrollo ágil y modularidad.

  • En Carro (Raspberry Pi): Flask y Flask-SocketIO (Servidor en tiempo real), librerías para manejo de sensores como `Adafruit_VL53L0X` y `PySerial`.
  • En Laptop (Dashboard): Tkinter (Interfaz Gráfica), SocketIO Client, y Folium (Visualización de mapas GPS).

Ventajas y Aplicaciones Potenciales

Ventajas Clave del Diseño

  • Alcance Ilimitado: La dependencia de la red 4G permite que la operación se extienda mucho más allá del alcance típico de Wi-Fi o Bluetooth.
  • Bajo Costo: Componentes como la Pi Zero 2 W y el chasis simple mantienen el proyecto dentro de un presupuesto ajustado.
  • Tiempo Real: Baja latencia (100-500ms) para la telemetría, crucial para el monitoreo y la toma de decisiones.
  • Autonomía: El carro gestiona la navegación y la evitación de obstáculos por sí mismo.

Aplicaciones Potenciales

  • Proyecto Educativo: Ideal para la enseñanza de robótica, IoT y programación Python avanzada.
  • Vigilancia y Mapeo: Exploración y registro de datos en terrenos de difícil acceso o zonas industriales.
  • Logística: Pequeño delivery o transporte en campus universitarios o parques cerrados.
  • Investigación: Plataforma económica para probar nuevos algoritmos de navegación.
